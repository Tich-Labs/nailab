# Button

Primary action button using Nailab brand colors.

## Usage

```
<button class="btn-nailab-blue">Primary Action</button>
<button class="btn-nailab-purple">Secondary Action</button>
```

## States
- Default
- Hover
- Disabled (add `opacity-50 cursor-not-allowed`)

## Tailwind Example
```
<button class="bg-nailabBlue text-white rounded-md px-5 py-2 font-semibold hover:bg-nailabBlue-dark">Primary</button>
```
