# Nailab Design System

This design system provides the foundational UX/UI guidelines and reusable components for Nailab-branded applications. It ensures visual consistency, accessibility, and ease of development across projects.

## Contents
- Design Tokens (colors, typography, spacing)
- Component Library (buttons, badges, cards, etc.)
- Usage Guidelines

## How to Use
- Reference the design tokens for consistent colors, fonts, and spacing.
- Use the provided component code or Tailwind classes for rapid development.
- Follow the guidelines for accessibility and responsive design.

---

For details, see the `tokens.md` and `components/` folder.
