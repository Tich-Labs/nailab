# Future Features for Nailab Admin

This document lists features and improvements to consider for future development. Add ideas, enhancements, and technical debt here for later prioritization.

## Planned Features

- Admin theme toggle (light/dark mode switch)
- Real-time notifications and notification center
- Admin user profile and avatar management
- Advanced search and filtering for admin data
- Audit logs and activity tracking
- Role-based access control and permissions
- Bulk actions for admin tables
- Improved dashboard analytics and charts
- In-app help and documentation
- Mobile-friendly admin interface

## Technical Improvements

- Refactor notification badge to use live data
- Integrate with external notification services (e.g., email, Slack)
- Accessibility improvements (ARIA, keyboard navigation)
- Performance optimizations for large datasets
