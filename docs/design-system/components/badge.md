# Badge

Status or label badge using Nailab brand colors.

## Usage

```
<span class="badge-nailab-blue">Info</span>
<span class="badge-nailab-purple">New</span>
```

## Tailwind Example

```
<span class="bg-nailabBlue-light text-nailabBlue-dark rounded-full px-3 py-1 text-sm font-medium">Info</span>
```
